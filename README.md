# telegram-coffee-text-analysis-bot
## Telegram bot for an AI generated text analysis model
This project uses a Trinoculars - an experimental AI detection tool for russian corpora texts.

- To use this wrapper, you need to download Trinoculars or Binoculars model and add a path in a config.py.

- To run a bot: 1) cd path\to\coffee-bot; 2) python -m trinoculars_telegram_bot.main or python trinoculars_telegram_bot/main.py
